Adventures in Indenting file for learning

Start with an Ordered List:

  1. This is one
  2. This is line two
  3. This is line three
  4. This is line four
  5. This is line last

 

Now we will add some indented:
  1. This is one
    • This is line two
    • This is line three
    • This is line four
  2. This is line five
  3. This is line six

 

Indented without bullets:
  1. This is one
    • This is line two
    • This is line three
    • This is line four
  2. This is line five
  3. This is line six
